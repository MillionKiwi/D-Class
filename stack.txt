프로젝트 기술 스택

==========================================
📱 프론트엔드 (사용자: P1, P2, P3)
==========================================

기본 프레임워크: Vue.js 3 (SPA)
- Composition API 사용
- Mobile-first PWA 구축
- 강사 및 학원 회원이 사용하는 모바일 앱

핵심 라이브러리:
- Pinia: 상태 관리 (Vuex 대체)
- Vue Router: SPA 라우팅
- Vite: 빌드 도구 및 개발 서버
- PWA 플러그인: Service Worker, 오프라인 지원

UI 프레임워크 (선택):
- Vuetify 또는 Quasar: 모바일 최적화 컴포넌트

지도 라이브러리:
- Kakao Map API: 한국 서비스 최적화

==========================================
🔧 백엔드 (API 서버)
==========================================

기본 프레임워크: Django 4.x + Django Rest Framework (DRF)
- Vue.js에 필요한 모든 데이터와 비즈니스 로직을 REST API로 제공

핵심 패키지:
- djangorestframework-simplejwt: JWT 인증
- django-cors-headers: CORS 설정
- django-filter: 필터링 기능
- django-storages: AWS S3 파일 저장소 연동
- django-celery-beat: 스케줄링 (선택)

==========================================
👨‍💼 백엔드 (관리자: P4)
==========================================

Django Admin
- P4의 모든 관리자 기능(회원 관리, 인증 검토 등)은 Django Admin을 통해 자동 생성하여 개발 공수를 최소화합니다.
- 커스터마이징: ModelAdmin, 커스텀 액션, 대시보드

==========================================
💾 데이터베이스
==========================================

PostgreSQL 14+
- 모든 데이터를 안정적으로 저장하는 관계형 데이터베이스(RDBMS)
- PostGIS 확장: 지도 기능 지원 (지리적 좌표, 반경 검색, 클러스터링)
- Full-Text Search: 공고 검색 최적화

==========================================
🔐 인증 방식
==========================================

JWT (JSON Web Token)
- 로그인 시 발급된 Access Token을 Authorization: Bearer <token> 헤더에 담아 전송합니다.
- Access Token: 15분 유효
- Refresh Token: 7일 유효 (HttpOnly Cookie 권장)
- 보안 강화: Rate Limiting, 토큰 블랙리스트 관리

==========================================
🚀 인프라 및 미들웨어
==========================================

필수 인프라:
- Redis: 캐싱, 세션 관리, Rate Limiting, 토큰 블랙리스트
- Celery: 비동기 작업 (파일 처리, 이메일 발송, 알림 발송)
- AWS S3: 파일 저장소 (인증 서류, 프로필 이미지)
- Nginx: 리버스 프록시, 정적 파일 서빙

선택 인프라:
- Django Channels: 실시간 알림 (WebSocket)
- Elasticsearch: 고급 검색 기능 (필요 시)
- RabbitMQ: Celery 메시지 브로커 (Redis 대안)

==========================================
📦 개발 도구 및 유틸리티
==========================================

프론트엔드:
- TypeScript: 타입 안정성 (선택, 대규모 프로젝트 권장)
- VueUse: 유틸리티 함수 모음
- Axios: HTTP 클라이언트

백엔드:
- pytest: 테스트 프레임워크
- black: 코드 포맷팅
- flake8: 린팅

모니터링 및 로깅 (선택):
- Sentry: 에러 모니터링
- Prometheus + Grafana: 성능 모니터링

==========================================
🔒 보안 고려사항
==========================================

- HTTPS 필수
- CORS 설정 (프론트엔드 도메인만 허용)
- Rate Limiting (API 호출 제한)
- CSRF 보호 (Django 기본 제공)
- XSS 방지 (Django 기본 제공)
- SQL Injection 방지 (ORM 사용)
- 파일 업로드 검증 (크기, 형식, 보안 스캔)
