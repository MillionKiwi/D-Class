# D-Class 테스트 시나리오

## 📋 문서 정보
- **프로젝트**: D-Class (디클래스) Frontend
- **버전**: 1.0.0
- **작성일**: 2025년 10월
- **목적**: MSW 기반 시나리오 테스트

---

## 🧪 테스트 환경 설정

### 사전 준비
1. 개발 서버 실행: `npm run dev`
2. MSW가 자동으로 활성화됨 (콘솔에 "✅ MSW가 활성화되었습니다." 메시지 확인)
3. 브라우저 개발자 도구를 열어 네트워크 탭 모니터링

### 테스트 계정
```javascript
// 강사 계정
Email: instructor1@test.com
Password: test1234

// 학원 계정  
Email: academy1@test.com
Password: test1234

// 관리자 계정
Email: admin@dclass.com
Password: test1234
```

## 📝 시나리오 1: 강사 회원 전체 플로우

### 1.1 회원가입 및 로그인
**목표**: 강사로 회원가입하고 로그인하기

**테스트 단계**:
1. 로그인 페이지 접속 (`/login`)
2. "회원가입" 버튼 클릭
3. 역할 선택 페이지에서 "강사" 선택
4. 회원가입 정보 입력:
   - 이메일: `test-instructor@test.com`
   - 비밀번호: `test1234`
   - 이름: `테스트강사`
   - 전화번호: `010-1111-2222`
   - 이용약관 동의
5. 회원가입 완료 후 로그인 페이지로 이동
6. 로그인 (기존 계정 사용: `instructor1@test.com` / `test1234`)

**예상 결과**:
- ✅ 회원가입 성공 토스트 메시지
- ✅ 로그인 페이지로 자동 이동
- ✅ 로그인 후 공고 목록 페이지로 이동
- ✅ 헤더에 사용자 이름 표시

---

### 1.2 공고 탐색 및 필터링
**목표**: 원하는 조건의 공고 찾기

**테스트 단계**:
1. 메인 대시보드 (공고 목록) 확인
2. 지역 필터 선택: "서울 강남구"
3. 장르 필터 선택: "발레"
4. 정렬 기준 변경: "급여 높은 순"
5. 공고 카드 확인

**예상 결과**:
- ✅ 필터에 맞는 공고만 표시
- ✅ 급여 높은 순으로 정렬
- ✅ 각 공고 카드에 학원명, 급여, 장르 등 표시
- ✅ 인증된 학원에는 인증 배지 표시

---

### 1.3 공고 상세 조회 및 찜하기
**목표**: 관심 공고를 자세히 보고 찜하기

**테스트 단계**:
1. 공고 카드 클릭 (예: "발레 전임강사 모집")
2. 공고 상세 정보 확인:
   - 학원 정보
   - 모집 요강
   - 급여 정보
   - 근무 조건
3. 우측 상단 하트 아이콘 클릭 (찜하기)
4. 뒤로가기 후 다른 공고도 찜하기

**예상 결과**:
- ✅ 상세 정보가 잘 표시됨
- ✅ 찜하기 클릭 시 "찜 목록에 추가되었습니다." 토스트
- ✅ 하트 아이콘 색상 변경 (활성화)
- ✅ 하단 네비게이션 "찜한 공고"에 카운트 증가

---

### 1.4 공고 지원하기
**목표**: 마음에 드는 공고에 지원하기

**테스트 단계**:
1. 공고 상세 페이지에서 "지원하기" 버튼 클릭
2. 확인 모달에서 "지원하기" 확인
3. 지원 완료 후 버튼 상태 확인

**예상 결과**:
- ✅ "지원이 완료되었습니다." 토스트 메시지
- ✅ "지원하기" 버튼 → "지원 완료" (비활성화)
- ✅ 카드에 "지원 완료" 배지 표시

---

### 1.5 지원 현황 확인
**목표**: 내가 지원한 공고 확인하기

**테스트 단계**:
1. 하단 네비게이션에서 "지원 현황" 클릭
2. 지원한 공고 목록 확인
3. 지원 상태별 필터 (전체/검토중/수락/거절)
4. 지원 항목 클릭하여 상세 정보 확인

**예상 결과**:
- ✅ 지원한 공고 목록 표시
- ✅ 각 공고의 지원 상태 표시
- ✅ 상태별 색상 구분
- ✅ 지원 날짜 표시

---

### 1.6 찜한 공고 관리
**목표**: 찜한 공고 보기 및 찜 해제

**테스트 단계**:
1. 하단 네비게이션에서 "찜한 공고" 클릭
2. 찜한 공고 목록 확인
3. 공고 카드의 하트 아이콘 클릭 (찜 해제)

**예상 결과**:
- ✅ 찜한 공고만 표시
- ✅ 찜 해제 시 목록에서 제거
- ✅ "찜 목록에서 제거되었습니다." 토스트

---

### 1.7 프로필 관리
**목표**: 강사 프로필 작성 및 수정

**테스트 단계**:
1. 하단 네비게이션에서 "마이페이지" 클릭
2. "프로필 관리" 클릭
3. 프로필 정보 입력/수정:
   - 자기소개
   - 전문 장르 선택
   - 경력 연수
4. 경력 추가:
   - 기관명: "서울발레단"
   - 직책: "무용수"
   - 기간 설정
5. 학력 추가:
   - 학교명: "서울예술대학교"
   - 전공: "무용학과"
   - 학위: "학사"
6. 저장

**예상 결과**:
- ✅ "프로필이 저장되었습니다." 토스트
- ✅ 입력한 정보가 잘 표시됨
- ✅ 경력/학력이 목록에 추가됨

---

### 1.8 인증 서류 제출
**목표**: 강사 인증 신청하기

**테스트 단계**:
1. 마이페이지에서 "인증 관리" 클릭
2. "학력/경력 인증" 선택
3. 서류 업로드 (이미지 파일 선택)
4. "제출하기" 버튼 클릭

**예상 결과**:
- ✅ "인증 신청이 완료되었습니다." 메시지
- ✅ 인증 상태 "대기중"으로 표시
- ✅ 업로드한 파일 목록 표시

---

### 1.9 알림 확인
**목표**: 알림 확인 및 관리

**테스트 단계**:
1. 헤더의 알림 아이콘 클릭 (빨간 점 표시)
2. 알림 목록 확인

- ✅ 알림 종류별 아이콘/색상
- ✅ 클릭 시 읽음 처리
- ✅ 관련 페이지로 이동

## 📝 시나리오 2: 학원 회원 전체 플로우

### 2.2 공고 등록
**목표**: 새로운 구인 공고 작성하기

**테스트 단계**:
1. 헤더의 "+" 버튼 클릭
2. 공고 정보 입력:
   - 제목: "재즈댄스 전임강사 모집"
   - 지역: "서울 강남구"
   - 장르: "재즈댄스"
   - 근무일: 월, 수, 금
   - 근무시간: 14:00 ~ 21:00
   - 급여: 월 280만원
   - 상세 설명 작성
   - 우대사항 입력
3. "등록하기" 버튼 클릭

**예상 결과**:
- ✅ "공고가 등록되었습니다. 관리자 검토 후 게시됩니다." 메시지
- ✅ 공고 관리 페이지로 이동
- ✅ 등록한 공고가 "검토 대기" 상태로 표시

---

### 2.3 공고 관리
**목표**: 등록한 공고 확인 및 수정

**테스트 단계**:

2. 등록한 공고 목록 확인
3. 공고 상태별 필터 (전체/진행중/마감)
4. 공고 카드의 "수정" 버튼 클릭
5. 급여 정보 수정: 월 300만원
6. "저장" 버튼 클릭

**예상 결과**:
- ✅ 등록한 공고 목록 표시
- ✅ 각 공고의 조회수, 지원자 수 표시
- ✅ "공고가 수정되었습니다." 토스트
- ✅ 수정된 정보 반영

5. 리뷰 작성:
   - 평점: 5점
   - 장점 입력
   - 단점 입력
   - 상세 후기 작성
6. "등록" 버튼 클릭

**예상 결과**:
- ✅ 채용 확정된 학원만 목록에 표시
- ✅ "리뷰가 등록되었습니다." 토스트
- ✅ 작성한 리뷰 목록에 추가
- ✅ 학원 프로필에 리뷰 반영
